# College Help Desk System

A LAN-based help desk system for students and staff.

## Features
- Student ticket submission
- Staff/Admin ticket management
- Role-based authentication (JWT)
- Real-time messaging per ticket
- Secure API with Node.js & PostgreSQL

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: PostgreSQL
- Auth: JWT
- Deployment: LAN Server

## How to Run

### Backend
```bash
cd Help-Desk-server
npm install
npm run dev
